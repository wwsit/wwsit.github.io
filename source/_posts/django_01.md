### 1. 创建虚拟环境

    mkvirtualenv python_django -p python3

### 2. 安装Django

    pip install django==1.11.11

### 3.虚拟环境和pip的命令

**虚拟环境**

    mkvirtualenv  # 创建虚拟环境
    rmvirtualenv  # 删除虚拟环境
    workon  # 进入虚拟环境、查看所有虚拟环境
    deactivate  # 退出虚拟环境

**pip命令**

    pip install  # 安装依赖包
    pip uninstall  # 卸载依赖包
    pip list  # 查看已安装的依赖库
